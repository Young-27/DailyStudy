1. HTML 태그분석
 1) input type="text"
-아이디 입력란
<input type="text" id="id" name="id" class="int" title="ID" maxlength="20">
-이름
<input type="text" id="name" name="name" title="이름" class="int" maxlength="40">
-이메일
<input type="text" id="email" name="email" placeholder="선택입력" aria-label="선택입력" class="int" maxlength="100">

- type="text" : 아이디, 이름, 이메일 입력란에는 input type 중 글자를 입력받는 text를 사용함 
- maxlength : 아이디, 이름, 이메일의 글자수가 적정한 수를 넘지 않도록 maxlength를 사용하여 글자수 제한을 둠
- placeholder : 이메일을 입력하는 란에는 placeholder를 사용하여 사용자가 클릭하기 전 "선택입력"이라는 문구가 보이도록 함

 2) input type="tel"
-전화번호 입력란
<input type="tel" id="phoneNo" name="phoneNo" placeholder="전화번호 입력" aria-label="전화번호 입력" class="int" maxlength="16">
- type="tel" : tel type을 사용해서 사용자가 전화번호 형식에 맞게 입력할 수 있도록 함

 3) input type="password"
-비밀번호 입력란 & 확인란
<input type="password" id="pswd1" name="pswd1" class="int" title="비밀번호 입력" aria-describedby="pswd1Msg" maxlength="20">
- type="password" : 문자입력 시 *로 표기되어 보안상 안전한 속성을 비밀번호 입력란에 사용함

 5) button
-가입하기 버튼
<button type="button" id="btnJoin" class="btn_type btn_primary"><span>가입하기</span></button>
- 입력 내용을 제출할 수 있도록 가입하기 버튼을 사용하였다.

-인증번호받기 버튼
<a href="#" class="btn_verify btn_primary" id="btnSend" role="button">
    <span class="">인증번호 받기</span>
</a>
- 인증번호받기 버튼을 누르면 인증번호를 받을 수 있도록 하이퍼링크를 사용하였다.



2. CSS 태그 분석
- NAVER 로고부분을 position:relative로 설정해 놓았다.
- 각 입력칸을 border: solid 1px #dadada; 로 테두리가 밝은 회색으로 보이도록 태그를 사용하였다.
- 비밀번호 자물쇠모양
    -position: absolute 태그를 사용하여 오른쪽구석에 뜨도록 고정시켜 놓았다.
    -background-image: url(https://static.nid.naver.com/images/ui/join/m_icon_pw_step.png);
     : 자물쇠 모양을 배경 이미지로 사용하도록 이미지 경로로 url을 제시해놓았다.
    -background-repeat: no-repeat;태그를 사용하여 이미지가 반복되지 않고 하나만 뜨도록 설정해놓았다.


-버튼 (인증번호받기, 가입하기)
- box-sizing: border-box; 태그를 사용하여 박스 테두리를 기준으로 크기를 설정하도록 했다.
- background-color: #03c75a; 태그를 사용하여 버튼부분을 네이버의 대표 색인 #03c75a으로 통일하였다.
